## **Ordinary Day：基于Unity-AP引擎开发的第一人称叙事冒险游戏**

### 一.**【简介】**

她的离世使我深受打击，记忆逐渐遗忘、扭曲（大脑保护机制）。但每天早晨起床都会延续梦中的痛，并不由自主地寻找那些自己曾藏起来的日记本。

但当找回所有蒙上灰尘的记忆后，我并不会更难过，反而心情平静。淡然地走出门，迎接平淡日常。

### **二.【说明】**

根据提示对物品进行交互，完成所有目标后结束游戏。
